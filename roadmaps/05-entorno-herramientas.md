# 🛠️ Módulo 05: Entorno y Herramientas

Un buen artesano conoce sus herramientas. Configura tu taller para ser más rápido y cometer menos errores.

## 📍 Temas a Dominar

### 1. Editor de Código (VS Code)
- **Atajos**: Domina el teclado para no depender del mouse.
- **Extensiones Clave**: Prettier (formateo), ESLint (errores), Live Server.

### 2. La Terminal
- **Navegación**: `cd`, `ls`, `mkdir`, `rm`, `cat`.
- **NPM / PNPM / Yarn**: Aprender a instalar paquetes y scripts externos.

### 3. Browser DevTools
- **Inspección de Elementos**: Revisar el HTML/CSS en tiempo real.
- **Console & Network**: Depurar JavaScript y revisar si las peticiones a la API funcionan.

## 🛠️ Retos Sugeridos
1.  Personalizar tu VS Code y aprenderte 5 atajos que no conocías.
2.  Instalar un paquete vía NPM (como `lodash` o `axios`) y usarlo en un pequeño script.
