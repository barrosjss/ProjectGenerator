# ⚙️ Módulo 03: Lógica y JavaScript

Aquí es donde las cosas se ponen interesantes. Dejamos de hacer "fotos" y empezamos a hacer aplicaciones que funcionan, piensan y reaccionan.

## 📍 Temas a Dominar

### 1. JavaScript Moderno (ES6+)
- **Sintaxis**: Let/Const, Arrow Functions, Template Literals.
- **Estructuras de Control**: If/Else, Switch, For/While, Array Methods (`map`, `filter`, `reduce`).
- **Desestructuración y Spread Operator**: Manipulación eficiente de objetos y arreglos.

### 2. Manipulación del DOM
- **Selección de Elementos**: `querySelector`, `getElementById`.
- **Eventos**: Escuchar clicks, inputs, scrolls y responder a ellos.
- **Creación de Elementos Dinámicos**: Modificar el HTML desde el JS.

### 3. Asincronía
- **Fetch API**: Cómo pedir datos a una base de datos o API externa.
- **Promises y Async/Await**: Manejar el tiempo y la espera de procesos.

## 🛠️ Retos Sugeridos
1.  Hacer un buscador de películas que consuma una API externa.
2.  Crear una lista de tareas (To-Do List) con persistencia en `localStorage`.
